Picture Gallery Application Documentation

This documentation provides an overview of the Picture Gallery Application, detailing its features, functionality, and how to use it. The application allows users to search for pictures based on a category name and retrieves relevant images from Unsplash.com using their API.

Introduction

The Picture Gallery Application is a web-based gallery that enables users to find and view images from Unsplash.com based on a category name. It demonstrates the developer's skills in JavaScript, CSS, HTML, and API integration. The application uses modern front-end technologies to provide a visually appealing and responsive user interface.

Features

The Picture Gallery Application offers the following features:

  • Text Input Field: Users can enter a category name in the text input field to search for relevant pictures.
  • Search Button: A search button is provided to initiate the picture search based on the entered category name.
  • Grid Display: The retrieved pictures are displayed in a responsive grid with a constant size, ensuring a visually consistent layout
  • Mobile Responsiveness: The gallery is designed to adapt its layout for optimal viewing on mobile devices, displaying the grid in a single column.

Requirements

To run the Picture Gallery Application, you need the following:

  • Internet Connection: The application requires an internet connection to make API calls to Unsplash.com and retrieve pictures.
  • Web Browser: Any modern web browser that supports HTML5, CSS3, and JavaScript is compatible with the application.
  • Unsplash API Key: To access the Unsplash.com API, you must sign up for a developer account and obtain an API key.

Usage

  • Upon opening the application in your web browser, you will see the Picture Gallery interface.
  • Enter a category name (e.g., "food" or "landscape") into the text input field.
  • Click the "Search" button to retrieve pictures related to the entered category from Unsplash.com.
  • The gallery will display the retrieved pictures in a grid format, with each image showing the author's name and description.
  • Double Click on image to download picture.
